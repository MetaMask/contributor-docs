# การมีส่วนร่วมกับโปรเจกต์ (Contributing)

## เป้าหมายของ Repository นี้คืออะไร?
Repository นี้เปรียบเสมือนพื้นที่สำหรับทีมต่าง ๆ ในระบบนิเวศของ MetaMask เพื่อแชร์ หารือ และบันทึกหลักการ แนวปฏิบัติ และรูปแบบในการสร้าง บริหารจัดการ และเผยแพร่ผลิตภัณฑ์รวมถึงไลบรารีต่าง ๆ ของ MetaMask ให้เป็นไปในทิศทางเดียวกัน

## เอกสารประเภทไหนที่ควรอยู่ในนี้?
ข้อมูลใดก็ตามที่เป็นประโยชน์ต่อทีม MetaMask และมีความปลอดภัยที่จะเปิดเผยต่อสาธารณะ

## ฉันจะช่วยได้อย่างไร?
แม้ว่าเอกสารบางส่วนจะเขียนขึ้นโดยทีมงานเฉพาะทาง แต่เป้าหมายหลักคือการทำให้ทุกคนสามารถนำไปใช้ประโยชน์ได้ หากคุณพบว่ามีข้อมูลใดขาดหายไป หรือมีความเห็นที่แตกต่างในเนื้อหาที่มีอยู่ คุณสามารถเปิด **Pull Request (PR)** เพื่อเสนอการแก้ไขได้ทันที
